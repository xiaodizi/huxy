# Voice Recording

Voice Recording dictates text into Muxy. Open it with `Cmd+Shift+I` or the microphone button in the status bar.

Transcription uses macOS on-device speech recognition. Audio is not sent to Muxy servers.

## Controls

| Action | Shortcut |
| --- | --- |
| Finish | `Return` |
| Pause / Resume | `Space` |
| Cancel | `Esc` |

When finished, Muxy inserts the transcript where your cursor was before recording. If that target is gone, the transcript is copied to the clipboard.

## Settings

Open **Settings -> Recording**:

- **Press Return after inserting** sends the transcript immediately.
- **Language** lists only languages with an installed on-device speech model.

If no language appears, add a dictation language in **System Settings -> Keyboard -> Dictation**.
